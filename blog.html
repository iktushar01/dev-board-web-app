Question-1: What are the different ways to select an element in the DOM?

There are many type of way to select an element in the DOM. getElementById select element by unique ID. getElementsByClassName select by class name.getElementsByTagName select by tag name.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : sets the HTML content inside an element. innerTest : sets only the visible text inside an element. textContent : sets all text inside an element.

Question-3: What is event delegation in the DOM?

Event delegation is a method where parent element listens for events on its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling means when you click on a child element, the event moves up to its parent, grandparent, and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element using document.createElement("tagName"). Add it using appendChild(), prepend(), or insertBefore(). Remove it using remove() or removeChild()
